## 使用

Vercel 构建预览地址：[https://picw.vercel.app/](https://picw.vercel.app/)

## 安装

```bash
git clone https://github.com/Flysky12138/PicW.git
cd PicW
yarn
```

## 开发

```bash
git clone -b dev https://github.com/Flysky12138/PicW.git
cd PicW
yarn

git add .
yarn cz
git push
```
